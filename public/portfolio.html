  • Library Management System

    This Library Management System is a Python-based application that facilitates the management of books, members, and genres within a library setting. Users can perform a variety of tasks such as adding, deleting, and searching for books, as well as managing member details and borrowing activities. The system is designed for easy setup and use, requiring only Python and MySQL, and includes a straightforward menu-driven interface for seamless library operations.

  • Contact Management System

    • The Contact Management System allows users to manage their contacts through a series of straightforward steps. Users can add, edit, delete, and search for contacts, as well as export and import contact lists for easy management. Each operation is facilitated via a simple menu option and requires specific information like email addresses and phone numbers to ensure accurate contact management.

  • E-Commerce API

    • This e-commerce application utilizes Flask, SQLAlchemy, and Marshmallow to manage four main components: Customers, Customer Accounts, Products, and Orders. It features robust data models and relationships, offering RESTful API endpoints for effective management of customer information, product catalogs, and order processing. The application facilitates operations like order placement, stock management, and user authentication setup, with Marshmallow schemas providing data validation and serialization, ensuring secure and efficient database interactions.
